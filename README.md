### What I learned

What I learned from this challenge was how to create an image overlay that appears when the user hovers over it, as well as how to position an icon within the overlay. This challenge also taught me the importance of using width effectively instead of relying solely on margins to achieve responsiveness. By doing so, I was able to create a layout that adapts better to different screen sizes and avoids layout inconsistencies or unexpected spacing issues.
